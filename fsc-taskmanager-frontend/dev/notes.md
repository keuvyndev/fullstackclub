Bibliotecas usadas

- yarn: permite fazer requisições http:

```bash
yarn i @latest
yarn start (inicializa a aplicação)
```

- axios: permite fazer requisições http:

```bash
yarn add axios
import axios from "axios"
```

- cors: para resolver o problema do CORS policy (instalado no back-end)

```bash
yarn add axios
```
